# 🚆 Railway Station Console Application (Python)

This is a console-based Railway Station Management System built using Python.

It allows users to view train details, schedules, platform information, and check train status.

---

## 📌 Project Description

This project simulates a railway station information system in the console.

Users can:

- View train names and numbers
- View train schedules
- Check platform status
- Search train details by train number

---

## 🛠️ Technologies Used

- Python 3
- Dictionary Data Structure
- While Loop
- Conditional Statements
- Datetime Module

---

## 📂 Features

✅ Displays current date and time  
✅ Menu-driven interface  
✅ Train schedule display  
✅ Platform information  
✅ Train search by number  
✅ Error handling for invalid options
